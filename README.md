
# Creating a reusable UI library for React

## Uses 

- https://rollupjs.org/
- https://storybook.js.org/
- https://jestjs.io/

npm run test

npm run storybook

npm run build

npm version minor
npm publish

## Run Storybook from a Docker Container

docker build -t react-ui-library .
docker run -p 80:8000 -d react-ui-library
